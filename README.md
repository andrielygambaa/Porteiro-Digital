# 🚪 Porteiro Digital

Bem-vindo ao projeto Porteiro Digital!

O Porteiro Digital é um sistema desenvolvido para realizar o controle de acesso em eventos, verificando automaticamente se o participante possui 18 anos ou mais.

## 🎯 Objetivo

Garantir que apenas pessoas maiores de idade tenham acesso a eventos restritos, utilizando verificação simples e automatizada.

## ⚙️ Funcionalidades

- Inserção da idade do participante
- Verificação automática da maioridade
- Mensagem de:
  - ✅ Acesso Permitido
  - ❌ Acesso Negado
- Interface simples e intuitiva

## 💻 Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## 📌 Regras do Sistema

- Idade menor que 18 anos → Acesso Negado
- Idade igual ou maior que 18 anos → Acesso Permitido
